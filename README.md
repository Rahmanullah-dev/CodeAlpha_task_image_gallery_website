# CodeAlpha_task_image_gallery_website
# 📷 PixelPerfect Gallery

![PixelPerfect Gallery Preview](https://via.placeholder.com/1200x600/1e293b/ffffff?text=PixelPerfect+Gallery)

A stunning, fully responsive image gallery website featuring an automatic background slider, categorized photo collections, and an interactive lightbox viewer. Built with vanilla HTML, CSS, and JavaScript.

## ✨ Live Demo

[View Live Demo](#) *(https://rahmanullah-dev.github.io/CodeAlpha_task_image_gallery_website/)*

## 📋 Features

### 🎯 Core Features
- **Automatic Background Slider** - 5 high-quality background images with smooth transitions and zoom animations
- **Responsive Gallery Grid** - 12 curated images with beautiful card layouts
- **Category Filtering** - Filter images by Nature 🌿, Urban 🏙️, and Portrait 👤 categories
- **Interactive Lightbox** - Click any image for full-screen viewing with navigation controls
- **Mobile-First Design** - Fully responsive across all devices
- **Smooth Navigation** - Fixed navbar with smooth scrolling to sections

### 🎨 Visual Highlights
- Gradient text effects and modern UI design
- Professional color scheme with warm gradients
- Hover animations on cards and buttons
- Glassmorphism effect on navbar
- Dynamic active states and transitions

### 📱 Interactive Elements
- Collection cards that filter gallery content
- Contact form with validation
- Mobile hamburger menu
- Active navigation highlighting on scroll
- Keyboard navigation support in lightbox

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Custom styling with animations and gradients
- **Vanilla JavaScript** - No external libraries required
- **Font Awesome** - Icons and visual elements
- **Unsplash/Picsum** - Placeholder images (easily replaceable)
